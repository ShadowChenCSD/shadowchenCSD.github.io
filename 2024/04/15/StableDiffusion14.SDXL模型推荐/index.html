StableDiffusion14.SDXL模型推荐

更新于2024-04-22|Stable Diffusion样图
|字数总计:670|阅读时长:2分钟|阅读量:

1. 优势

1.1 提示词兼容性大幅提升

  • 不再强制需要 masterpiece、high quality、best quality 等提示词来提升画面质量

  • 对自然语言的理解更加到位，逐渐从词条化方案转变为自然语言表述的方案

1.2 更大、更高清的分辨率

  • SD1.5时代使用的训练样本数据，尺寸多为512像素、768像素的正方形图片

  • SDXL时代使用的训练样本数据，尺寸提升到1024像素的正方形图片，可以直接画出分辨率更大的图片

  • 判断方法：在使用更大的分辨率作图时，AI会误以为是在生成多张图片，从而产生多人的现象

1.3 图片精度大幅提升

  • 可以使用Refiner模型进行图生图

1.4 生成准确度大幅提升

  • 会写字：具有初步书写文字的能力

  • 会画手：手指错乱的情况有所降低

1.5 风格自由度大幅提升

  • 默认模型风格：接近于照片风真实系的效果
  • 官方的SDXL模型支持十多种画面风格

2. 劣势

2.1 通用模型，非定制化开发

  • 官方的SDXL是一个基础模型，没有经过微调，也不是为了满足某一个特定需求而量身定制

2.2 LoRA兼容性问题

  • 可能不兼容SD1.5版本训练出来的LoRA模型

3. 模型推荐

SDXL

官方基础模型

  • sd_xl_base_1.0.safetensors：用于文生图功能，根据文字描述，生成样图
  • sd_xl_vae.safetensors：获得更鲜艳的颜色和更清晰的图像
  • sd_xl_refiner_1.0.safetensors：用于图生图功能，对输入图像作细节强化

原理

推荐出图的分辨率

  • 【21:9】 1536 x 640
  • 【16:9】 1344 x 768
  • 【3:2】 1216 x 832
  • 【5:4】 1152 x 896
  • 【1:1】 1024 x 1024

  4. 迭代步数：35-50
