StableDiffusion13.LoRA模型推荐

更新于2024-06-28|Stable Diffusion 样图
|字数总计:393|阅读时长:1分钟|阅读量:

1. 基本概念

全称：Low-Rank Adaptation Model

意义：降低模型训练的门槛，拓宽产出模型的适用范围

文件格式：ckpt、safetensors

2. 加载方式

方式一：直接在提示词里输入

  • 模型存放位置：StableDiffusionWebUI/models/Lora
  • 格式：<Lora:LoRA模型文件名:权重>

方式二：附加模型选单

  • 缩略图：与模型同名的PNG图像

方式三：附加网络扩展

  • 扩展名称：Additional Network
  • 最大支持数目：5个Lora模型

3. 模型推荐

角色类

云韵 《斗破苍穹》

  • 底模：激萌v2国漫3D大模型

纳西妲《原神》

  • 底模：AnythingV5、cuteYukiMix

物品类

盲盒：blind box

  • 底模：pvc手办风格转换模型

  • 推荐尺寸：1:1、2:3、3:4、9:16

汝瓷

  • 底模：ReVAnimated

刺绣

  • 底模：RealisticVision
  • 关键词强化：simple background（简单背景）, embroidery（刺绣）, no humans（没有人）, bird, leaf, traditional media（传统媒体）
  • Lora权重：0.4 - 0.8
  • 推荐分辨率：512 * 768 、768 * 768、512 * 1024

服饰类

汉服

  • 底模：真实系模型
  • 提示词：han style、tang style、 song style、 ming style、 jin style
  • Lora权重：0.5至0.7

画风类

线稿：lineart

  • 底模：AnythingV5
  • 高分修复模型：R-ESRGAN 4x+ Anime6B
  • 提示词：可以用greyscale来强行设置为黑白成像

沁彩：Colorwater_v4.0

  • 底模：AnythingV5
  • 高分修复模型：Latent
  • LoRA权重：0.8 左右

墨心：MoXin

  • 底模：AnythingV5、CetusMix
  • LoRA权重：0.4 左右

Q版手绘

  • 底模：CetusMix、GhostMix
  • 推荐尺寸：512 * 512（放大1.5至2倍）、785 * 512
  • LoRA权重：0.8 左右

吉卜力风格：Studio Ghibli Style

  • 底模：AnythingV5、CetusMix
  • LoRA权重：0.8 左右
