StableDiffusion10.ControlNet插件

更新于2024-04-07|Stable Diffusion使用
|字数总计:899|阅读时长:2分钟|阅读量:

1. 基本原理

本质：Stable Diffusion的扩展插件

定位：对大扩散模型做微调的额外网络

作用：基于一些额外输入给它的信息，来给扩散模型的生成，提供明确的指引

2. 安装

推荐安装方式：解压缩包直接安装

  • 把ControlNet扩展文件夹放进根目录下的extensions文件夹内

  • webUI刷新插件

模型下载

  • 单个控制模型的大小在1.4G左右

  • 最新版本一共有十几个模型

  • 需要配合同名的yaml文件一起使用

  • 地址：huggingface.co/lllyasviel/ControlNet-v1-1

  • 存放：StableDiffusionWebUI/models/ControlNet

其他安装方式：需要额外手动下载模型

  • 通过仓库链接安装

  • 通过扩展列表搜索，直接下载ControlNet

3. 使用

  • 导入参考图（原图或信息图）
  • 勾选“启用”
  • 配置“预处理器”
  • 配置“模型”

4. ControlNet参数

  • 控制权重：默认为1

  • 开始控制阶段：0.0 - 1.0

  • 结束控制阶段：0.0 - 1.0

  • 控制模式：平衡、偏重提示词、偏重ControlNet

  • 缩放模式：拉伸、裁剪、填充

  • 反复迭代：默认关闭

  • 预控制器参数：每种预处理器的参数不尽相同

  • 其他参数

    • 低显存模式：适用于显存不够的时候，代价是出图速度变慢

    • Pixel Perfect：自动计算预处理器产出图像的最佳分辨率，推荐使用

    • Allow Preview：会打开另外的小预览器窗口，专门为预处理图像准备的

  • 信息图：需要手动保存，系统不会自动保存

5. 推荐模型

动作姿势：OpenPose

  • 骨骼图：控制人物体态的呈现

  • OpenPose：五官（鼻子、眼睛）、四肢

  • OpenPose-Hand：添加对手部骨骼的精细刻画
  • OpenPose-Face：添加面部特征
  • OpenPose-FaceOnly：只关注面部特征
  • OpenPose-Full：包含OpenPose、OpenPose-Hand、OpenPose-Face等功能

深度：Depth

  • 对场景的描绘还原，尤其是富有空间感的多层次场景
  • 深度图：越白的地方离得越近，越黑的地方离得越远
  • 预处理器
    • Depth Leres
    • Depth Leres++（效果最好）
    • Depth Midas
    • Depth Zoe
  • 如果OpenPose无法准确识别动作，尝试使用Depth，它可以更好地区分前后位置

边缘检测：Canny

  • 控制物体轮廓，还原图像外形特征
  • Canny低阈值：识别暗部
  • Canny高阈值：识别亮部
  • 线条不要太过密集，不然生成的图像可能有很多无用的细节
  • 应用
    • 线稿上色
    • 预处理器需要设定为：反转
    • LineArt：专门基于动漫风格的线稿，实现上色功能的预处理器和模型组合

柔和边缘：SoftEdge

  • 整体边缘线条检测（Holistically-Nested Edge Detection）

  • 效果和Canny算法类似，但边缘效果更为柔和——给AI更多的自由度

其他

  • inPaint模型：实现创新性局部重绘
  • Tile模型：在放大工作流中用于增加细节
  • Reference Only模型：用于固定特征
  • mlsd模型：识别直线，用于建筑物设计上
  • scribble模型：涂鸦
  • shuffle模型：画风转移
  • normal模型：参考原图的明暗关系，保持原图的姿势
  • ip2p模型：给照片加特效

6. 多重ControlNet应用

  • 步骤
    1. 在设置选项中，找到ControlNet
    2. 增大多重ControlNet的最大模型数量，默认为1
  • 意义：实现模型组合功能
  • 举例
    • OpenPose + Depth（权重：1、0.6）：控制人物姿势和空间信息
    • Depth + Canny：强化空间和空间细节信息
    • OpenPose + SoftEdge：控制人物姿势在大轮廓下的正确性
